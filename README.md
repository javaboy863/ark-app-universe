# 什么是ark宇宙
&emsp;&emsp;Ark宇宙是我个人在十多年职业生涯中，个人成果的结晶。包括：
 - 个人写的中间件
 - 个人沉淀的架构（包括应用架构/技术架构/业务架构/数据架构）方面的实践和方法论
 - 个人写的一些平台服务
 - 个人沉淀的中间件/组件的最佳实践
 - 软件全生命周期过程中涉及的各种规范
<br>
希望能够帮助到大家，如果有什么问题，欢迎提issue，我会尽快回复。
